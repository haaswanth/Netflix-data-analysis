# Netflix Data Analysis

## Project Overview
This project analyzes the Netflix titles dataset to understand content distribution, trends, and patterns using Python.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Dataset
Netflix Titles dataset containing information about movies and TV shows available on Netflix.

## Analysis Performed
1. Distribution of Movies and TV Shows on Netflix
2. Countries producing Netflix content
3. Most common ratings on Netflix
4. Year with highest number of content releases
5. How many Netflix titles are produced by each country

## Conclusion
The analysis provides insights into Netflix content trends, including content types, ratings, countries, and release patterns.
